Code Reference

Welcome to the NodeBox technical reference. Here you'll find a high-level overview, module documentation, and API details to help you extend or debug the project.

Architecture Overview

Main Application
File: main.py
Purpose: Launches UI, loads fonts, applies styles, starts Ollama
Automation Engine
Purpose: Handles workflow execution, node management, Ollama integration
Node System
Files: canvasmanager, ui/node.py
Purpose: Python node objects with inputs/outputs, visually connected on canvas
UI Components
Files: ui/main_window.py, node_editor_window.py
Purpose: PyQt6-based main window, node editor, model browser, automation manager
Ollama Integration
Files: main.py, models_data.py
Purpose: Manage local AI models, download, metadata

Key Modules

canvasmanager
Files: canvas_nodemethods.py, canvas_widget.py, ports_handler.py, saveload_methods.py
Purpose: Node logic, canvas UI, port handling, workflow save/load
ui
Files: main_window.py, node_editor_window.py, model_card.py, downloaded_models_window.py
Purpose: Main window, node editor, model display, downloaded models
utils
Files: font_loader.py, node_runner.py, paths.py
Purpose: Font management, node execution, path utilities
models_data.py
Purpose: Model metadata, supported models, icons, configuration

Class & Function Reference

main.py
Function: main()
Purpose: Launches the app, sets up UI, starts Ollama
Returns: None
Example: python main.py
canvasmanager.canvas_nodemethods
Function: add_node(type, position)
Purpose: Add node to canvas
Parameters: type (str), position (tuple)
Returns: Node object
Example: add_node('Input', (100, 200))
canvasmanager.canvas_nodemethods
Function: remove_node(node_id)
Purpose: Remove node by ID
Parameters: node_id (str)
Returns: None
ui.main_window
Class: MainWindow
Purpose: Main application window
Methods: show(), close(), load_automations(), browse models
utils.node_runner
Function: run_node(node)
Purpose: Executes node logic
Parameters: node (Node object)
Returns: Output data
models_data.py
Function: get_model_info(model_name)
Purpose: Get model metadata
Parameters: model_name (str)
Returns: dict
